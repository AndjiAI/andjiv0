# Getting Started

Welcome! Let's get you set up with andji, the AI coding agent.

## Installation

Install andji globally using npm:

```bash
  npm install -g andji
```

## Basic Usage

Start using andji in your project directory:

```bash
  andji
```

Or, start a new project with one of our [starter templates](https://github.com/andjiAI/andji-community/tree/main/starter-templates):

```bash
  andji --create nextjs project-name
```

That's it! andji is now ready to help you with your coding tasks.

## Troubleshooting Install

If you run into issues during installation:

1. **Check Node Version**

   - Make sure you have [Node version 18+](https://nodejs.org/en/download) installed
   - Verify your version: `node --version`

2. **Permission Issues**

   - On Mac/Linux: Try `sudo npm install -g andji`
   - On Windows: Run command prompt as administrator

3. **Still Having Problems?**
   - Try reinstalling Node using [nvm or fnm](https://nodejs.org/en/download)
   - Run the andji install command again

If you're still experiencing issues, check out our [Troubleshooting](/docs/advanced/troubleshooting) guide or join our [Discord community](https://andji.com/discord)!
